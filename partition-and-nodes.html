⚡ 關鍵概念

1️⃣ 單一 Node 的 Cores

定義：該 node 的 CPU 核心數（物理或邏輯核心）

作用：決定這個 node 能同時執行幾個 tasks

2️⃣ 集群總 Cores 數

定義：所有 nodes 的 cores 加總

作用：決定整個集群的最大並行度

3️⃣ Partitions 與 Cores 的關係

• Partitions > Cores：需要多輪執行，cores 會依序處理

• Partitions = Cores：最佳狀態，所有 cores 都在工作

• Partitions < Cores：資源浪費，部分 cores 閒置
